microIDE is a fully integrated Editor for the ESP32.
Visit mircoIDE.com for more details.
For some examples, visit the learning section
If you like the work please consider donating or supporting the development
Ussage:
Rescue: Triggers a Hardware Reset
Files: Shows the file directory.
Info: Opens the information page
Graph: Opens the Graphing Tool
Terminal: Opens the Terminal Monitor
New: Creates a new file in the editor
Close: Close the file
Open: Opens the file in the web browser
Edit: Edit the file in microIDE
Save: Saves the file to disk
Reload: Load file from disk
Run: Run the .py file on Disk
Delete: Delete the file completly
